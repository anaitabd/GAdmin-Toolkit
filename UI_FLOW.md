## Interaction Examples

### Single Email Flow:
1. User selects "Single Email" tab
2. Chooses method (API or SMTP)
3. Fills in sender email
4. If SMTP, enters password
5. Enters recipient email
6. Fills in from name, subject, HTML content
7. Clicks "Send Email"
8. System validates input
9. Sends email via selected method
10. Shows success/error message

### Bulk Email Flow:
1. User selects "Bulk Email" tab
2. Uploads users CSV file
3. System displays count of loaded users
4. Uploads recipients CSV file
5. System displays count of loaded recipients
6. Chooses method (API or SMTP)
7. Fills in from name, subject, HTML content
8. Clicks "Send Bulk Emails"
9. System distributes emails across sender accounts
10. Shows summary (sent/failed counts)

### Python Script Flow:
1. User selects "Python Script" tab
2. Ensures CSV files exist in files/ directory
3. Clicks "Execute Python Script"
4. System spawns Python process
5. Script reads CSV files and sends emails
6. Shows execution result and output

## CSS Styling

### Colors Used:
- Primary Blue: #4285f4 (buttons, active states)
- Success Green: #34a853 (success messages)
- Error Red: #ea4335 (error messages)
- Gray Text: #5f6368 (labels, secondary text)
- Light Gray: #f8f9fa (upload sections)
- White: #ffffff (cards, forms)

### Layout:
- Max width: 1200px (centered container)
- Card style: White background with shadow
- Border radius: 8px
- Padding: 20-24px
- Font: System fonts (Segoe UI, Roboto, etc.)

## Responsive Design

The UI is responsive and adapts to different screen sizes:
- Desktop: Full width with side-by-side elements
- Tablet: Stacked elements with appropriate spacing
- Mobile: Single column layout with full-width inputs

## Accessibility Features

- Semantic HTML elements
- Form labels for all inputs
- Clear error messages
- Keyboard navigation support
- Focus indicators on interactive elements
- Loading states for async operations
